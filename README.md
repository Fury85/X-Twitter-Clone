X (Twitter) Clone

A fully responsive X (Twitter) clone built using HTML, Tailwind CSS and JavaScript.

Features
Fully responsive layout
Responsive sidebar and bottom navigation
Dynamic active navigation states
Home feed UI
Explore page
Notifications page
Chat interface
Follow suggestions page
Profile page
Responsive post cards
Reusable utility and component classes
Mobile, tablet, and desktop support
Tech Stack
HTML5
Tailwind CSS
JavaScript
